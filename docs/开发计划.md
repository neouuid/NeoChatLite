## 技术决策记录

### 2026-04-10
- **决策**: 使用 React Native 而非 Flutter
- **理由**:
  - WebRTC 生态更成熟 (react-native-webrtc)
  - React Native for Windows/macOS 由微软官方维护
  - 与现有 Web 技术栈衔接更好

### 2026-04-10
- **决策**: 采用 shared + mobile + desktop 三层架构
- **理由**: 最大化代码复用 (85-90%)，同时保持各平台特性
