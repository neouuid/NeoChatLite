## 快速开始

### 后端

```bash
cd backend

# 使用 Docker Compose 启动数据库
docker-compose up -d

# 运行后端
go run ./cmd/api-gateway
```

### 前端 (移动端)

```bash
cd client/mobile

# 安装依赖
npm install

# 启动 Metro
npm start

# 运行 iOS
npm run ios

# 运行 Android
npm run android
```

### 前端 (桌面端)

```bash
cd client/desktop

# 安装依赖
npm install

# 启动 Windows
npm run windows

# 启动 macOS
npm run macos
```
