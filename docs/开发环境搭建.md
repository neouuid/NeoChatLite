## 后端环境搭建

### 1. 安装 Go

#### macOS

```bash
# 使用 Homebrew 安装
brew install go

# 验证安装
go version
```

#### Windows

```powershell
# 使用 winget 安装
winget install GoLang.Go

# 或从官网下载安装包
# https://golang.org/dl/

# 验证安装
go version
```

#### Linux

```bash
# Ubuntu / Debian
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz

# 添加到 PATH
echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
source ~/.bashrc

# 验证安装
go version
```

### 2. 配置 Go 环境

```bash
# 设置 GOPROXY (国内加速)
go env -w GOPROXY=https://goproxy.cn,direct

# 设置 GO111MODULE
go env -w GO111MODULE=on

# 查看配置
go env
```

### 3. 克隆项目

```bash
git clone git@github.com:neouuid/NeoChatLite.git
cd NeoChatLite
```

### 4. 安装依赖

```bash
cd backend

# 下载依赖
go mod download

# 验证依赖
go mod tidy
```

### 5. 运行后端

```bash
# 开发模式运行
cd cmd/api-gateway
go run main.go

# 或使用 air 热重载 (推荐)
# 先安装 air
go install github.com/cosmtrek/air@latest

# 然后运行
air
```

后端服务将在 `http://localhost:8080` 启动。

## 前端环境搭建

### 1. 安装 Node.js

#### macOS

```bash
# 使用 Homebrew 安装
brew install node@20

# 验证安装
node --version
npm --version
```

#### Windows

```powershell
# 使用 winget 安装
winget install OpenJS.NodeJS.LTS

# 或从官网下载安装包
# https://nodejs.org/

# 验证安装
node --version
npm --version
```

#### Linux

```bash
# 使用 NVM 安装 (推荐)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
source ~/.bashrc

# 安装 Node.js 20
nvm install 20
nvm use 20

# 验证安装
node --version
npm --version
```

### 2. 配置 npm (可选)

```bash
# 设置国内镜像源 (国内推荐)
npm config set registry https://registry.npmmirror.com

# 验证配置
npm config get registry
```

### 3. 安装项目依赖

```bash
cd client

# 安装根目录依赖
npm install

# 安装 shared 依赖
cd shared
npm install

# 安装 mobile 依赖
cd ../mobile
npm install

# 安装 desktop 依赖
cd ../desktop
npm install
```

## 移动端环境搭建

### iOS 开发环境 (仅 macOS)

#### 1. 安装 Xcode

从 Mac App Store 安装 Xcode 15+。

```bash
# 验证安装
xcodebuild -version

# 安装 Xcode Command Line Tools
xcode-select --install
```

#### 2. 安装 Cocoapods

```bash
# 使用 Homebrew 安装
brew install cocoapods

# 验证安装
pod --version
```

#### 3. 配置 iOS 项目

```bash
cd client/mobile

# 安装 iOS Pods
cd ios
pod install

# 返回项目根目录
cd ..
```

#### 4. 运行 iOS 模拟器

```bash
# 启动 Metro Bundler
npm start

# 在新终端运行 iOS 模拟器
npm run ios

# 或指定设备
npm run ios -- --simulator="iPhone 15 Pro"
```

### Android 开发环境

#### 1. 安装 Java JDK

```bash
# macOS
brew install --cask temurin17

# Windows
# 下载并安装: https://adoptium.net/

# Linux
sudo apt install openjdk-17-jdk

# 验证安装
java -version
```

#### 2. 安装 Android Studio

从官网下载安装 Android Studio: https://developer.android.com/studio

安装后，打开 Android Studio 并安装:
- Android SDK (API 33+)
- Android SDK Platform-Tools
- Android SDK Build-Tools
- Android Emulator

#### 3. 配置环境变量

```bash
# macOS / Linux (添加到 ~/.bashrc 或 ~/.zshrc)
export ANDROID_HOME=$HOME/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/emulator
export PATH=$PATH:$ANDROID_HOME/tools
export PATH=$PATH:$ANDROID_HOME/tools/bin
export PATH=$PATH:$ANDROID_HOME/platform-tools

# Windows (添加到系统环境变量)
# ANDROID_HOME=C:\Users\你的用户名\AppData\Local\Android\Sdk
# Path 中添加 %ANDROID_HOME%\platform-tools 等
```

#### 4. 创建 Android 模拟器

1. 打开 Android Studio
2. 进入 Tools → Device Manager
3. 点击 "Create Device"
4. 选择设备 (推荐 Pixel 6)
5. 选择系统镜像 (API 33+)
6. 完成创建

#### 5. 运行 Android 模拟器

```bash
cd client/mobile

# 启动 Metro Bundler
npm start

# 在新终端运行 Android 模拟器
npm run android
```

## Docker 开发环境

### 1. 启动后端依赖服务

```bash
cd backend/deployments/docker

# 启动所有服务
docker-compose up -d

# 查看服务状态
docker-compose ps

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down

# 停止服务并删除数据
docker-compose down -v
```

### 2. 服务说明

| 服务 | 地址 | 说明 |
|------|------|------|
| PostgreSQL | `localhost:5432` | 数据库 |
| Redis | `localhost:6379` | 缓存 |
| NATS | `localhost:4222` | 消息队列 |
| NATS UI | `http://localhost:8222` | NATS 管理界面 |

### 3. 数据库配置

默认连接信息:

```yaml
host: localhost
port: 5432
user: postgres
password: postgres
database: neochat
```

### 4. 连接数据库

使用 psql 连接:

```bash
docker exec -it neochat-postgres psql -U postgres -d neochat
```

或使用数据库管理工具 (DBeaver, DataGrip, TablePlus 等) 连接。

## 常见问题

### 后端问题

#### Q: go mod download 很慢或失败

A: 设置国内代理:

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```

#### Q: 端口被占用

A: 修改 `backend/configs/config.yaml` 中的端口，或停止占用端口的进程:

```bash
# Windows
netstat -ano | findstr :8080
taskkill /PID <进程ID> /F

# macOS / Linux
lsof -ti:8080 | xargs kill -9
```

---

### 前端问题

#### Q: npm install 很慢或失败

A: 设置国内镜像源:

```bash
npm config set registry https://registry.npmmirror.com
```

#### Q: Metro 缓存问题

A: 清除 Metro 缓存:

```bash
npm start -- --reset-cache
```

#### Q: iOS pod install 失败

A: 更新 CocoaPods 和 repo:

```bash
cd client/mobile/ios
pod repo update
pod install
```

---

### Android 问题

#### Q: Android 模拟器无法启动

A: 检查是否启用了虚拟化:

- Windows: 启用 Hyper-V 或 WHPX
- macOS: 不需要额外配置
- Linux: 启用 KVM

#### Q: Gradle 构建很慢

A: 配置 Gradle 镜像:

编辑 `client/mobile/android/build.gradle`:

```gradle
repositories {
    maven { url 'https://maven.aliyun.com/repository/google' }
    maven { url 'https://maven.aliyun.com/repository/jcenter' }
    maven { url 'https://maven.aliyun.com/repository/public' }
}
```

---

### Docker 问题

#### Q: Docker Compose 启动失败

A: 检查端口是否被占用:

```bash
docker-compose ps
docker-compose logs
```

#### Q: 数据库连接失败

A: 检查 PostgreSQL 容器状态:

```bash
docker ps
docker logs neochat-postgres
```
