## 设计概述

### 设计风格

- **设计语言**: 现代简约
- **主题**: 支持深色 / 浅色双主题
- **品牌色**: 紫色系 (#6366f1)
- **设计工具**: Pencil (.pen 文件)

### 平台适配原则

| 平台 | 设计特点 |
|------|---------|
| 移动端 | 单页面导航，底部 Tab，手势操作 |
| 桌面端 | 三栏布局，顶部菜单，鼠标操作 |
| Web | 响应式布局，适配各种屏幕 |

```typescript

  paddingHorizontal: 24,
  backgroundColor: COLORS.primary,

  color: '#ffffff',
  fontSize: TYPOGRAPHY.sizes.md,
  fontWeight: TYPOGRAPHY.weights.semibold,
```

## 动画规范

### 动画时长

| 类型 | 时长 (ms) |
|------|----------|
| 快速 | 150 |
| 标准 | 300 |
| 慢速 | 500 |

### 动画曲线

```typescript
Easing.bezier(0.25, 0.1, 0.25, 1) // 标准曲线
Easing.bezier(0.4, 0, 0.2, 1)    // 减速曲线
Easing.bezier(0.175, 0.885, 0.32, 1.275) // 弹性曲线
```

### 常见动画

#### 页面切换

```typescript
{
  duration: 300,
  easing: Easing.bezier(0.25, 0.1, 0.25, 1),
}
```

#### 淡入淡出

```typescript
{
  duration: 200,
  easing: Easing.inOut(Easing.quad),
}
```

#### 滑入滑出

```typescript
{
  duration: 250,
  easing: Easing.out(Easing.quad),
}
```
